Students will need the following:

- Python 3.8+
- Standard Python libraries (no external packages required)
- An IDE or text editor (e.g., VSCode, PyCharm)
- unittest (Python's built-in testing module)

There are no additional costs or account requirements associated with this project. Students can complete the project on their local machine or in a cloud-based Python environment.
